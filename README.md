# Pilgrimage

## Run Locally

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open the local URL shown in the terminal, usually:

```text
http://localhost:5173
```

## Build For Production

Create a production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Project Structure

- `media/` original photography used throughout the site
- `src/App.tsx` main one-page composition
- `src/components/` reusable editorial sections
- `src/data/chapters.ts` poem lines, image mapping, captions, and metadata
- `src/index.css` global visual system, typography, and mood styling
